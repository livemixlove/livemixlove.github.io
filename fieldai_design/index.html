Data Review Tool Boston Dynamics

Data Management • Review Interface • Workflow Optimization

Improvements

Designed a browser‑based review workspace that streams RGB + depth, overlays 3‑D detections, and lets users scrub events on a film‑strip timeline.

One‑click bookmarks and JSON annotations replace ad‑hoc spreadsheets, making hand‑offs between engineering and support painless.

Lightweight PNG‑depth pipeline keeps frame rates high—even on warehouse laptops.

Process

Shadowed pick‑station operators, distilled workflows into user stories, and iterated UI flows before writing code. On‑site trials fine‑tuned performance targets and interaction details.

Outcome

Previously cumbersome "find‑the‑failure" workflows are now simple and achievable by both low and high technical ability employees, customers, and application engineers. Much improved communication between client and installation teams. Positive feedback from sales team as well.

GTFS Manager Trillium Transit

Transit Data • GTFS • Data Management

Improvements

Replaced spreadsheet‑style editing with an interactive calendar matrix that shows trips, exceptions, and detours in one glance.

Inline validation flags feed errors before export; bulk actions (copy week, fill patterns) cut data‑entry time in half.

GraphQL diff‑patch backend supports multi‑user editing without overwriting each other's work.

Process

Interviewed planners across ten agencies, produced clickable prototypes, and iterated weekly alongside a small front‑end team. Rapid feature‑flag releases let early adopters shape the final product.

Outcome

More than a hundred agencies manage feeds with GTFS Manager; schedule updates now publish in minutes, not overnight batches, and validation failures have fallen sharply.

Core stack: React • Redux • GraphQL • PostgreSQL
