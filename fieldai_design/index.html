Data Visualization Tool

Agility Robotics
Data Visualization • UI Design • User Experience

Improvements

The Data Visualization Tool was redesigned to improve data clarity and user interaction. Key improvements included a more intuitive interface, enhanced chart readability, and streamlined navigation between different visualization types. The tool now offers customizable dashboards and real-time data filtering capabilities.

Process

The redesign process began with analyzing user pain points and identifying key opportunities for improvement. After conducting stakeholder interviews and usability testing, I developed wireframes and interactive prototypes that were iteratively refined based on user feedback. The final implementation resulted in a more powerful yet accessible visualization system.

Data Review Tool

Boston Dynamics
Data Management • Review Interface • Workflow Optimization

Improvements

The Data Review Tool was enhanced to streamline the data validation and review process. The redesign focused on improving data presentation, adding batch processing capabilities, and implementing a more efficient approval workflow. The tool now features error highlighting, comparison views, and automated quality checks.

Process

The development process started with shadowing data reviewers to understand their daily workflows and challenges. I created user journey maps to identify bottlenecks and conducted workshops to generate solution ideas. After prototyping key interactions, the design was validated through task-based testing with actual users, resulting in a more efficient review system.

GTFS Manager

Trillium Transit
Transit Data • GTFS • Data Management

Improvements

The GTFS Manager was redesigned to simplify the creation, editing, and validation of transit data. Key improvements included a visual route editor, automated validation tools, and a more intuitive schedule management interface. The tool now provides real-time feedback on data quality and compliance with GTFS specifications.

Process

The design process began with extensive research into GTFS data structure and workflows. I analyzed common errors in existing datasets and interviewed transit planners to understand their challenges. After mapping out the complex relationships in GTFS data, I developed a modular interface that breaks down the complexity while maintaining data integrity. The design went through multiple iterations of usability testing before implementation.
